报错一：

==Caused by: org.apache.hadoop.ipc.RemoteException(org.apache.hadoop.hdfs.server.namenode.SafeModeException): Cannot create directory /tmp/hive/wzj/1285d7ee-00f3-42a0-9161-03660fcb6a86. Name node is in safe mode.==

报错信息显示namenode进入安全模式，所以查看一下namenode的日志

==解决：==
既然namenode进入安全模式，那就让它退出安全模式不就行了

ok!已进入

报错二：

==hive (default)> show databases;
FAILED: SemanticException org.apache.hadoop.hive.ql.metadata.HiveException: java.lang.RuntimeException: Unable to instantiate org.apache.hadoop.hive.ql.metadata.SessionHiveMetaStoreClient
hive (default)>==

报错无法实例化，metadata有问题，ok,那就去看一下元数据的库

[mysqladmin@hadoop001 ~]$ service mysql status
ERROR! MySQL is not running, but lock file (/var/lock/subsys/mysql) exists

发现mysql的服务没有启，那就启动一下吧

[mysqladmin@hadoop001 ~]$ service mysql start
Starting MySQL… SUCCESS!
[mysqladmin@hadoop001 ~]$ service mysql status
SUCCESS! MySQL running (4136)

重新启动一下Hive吧

ok 没有问题
