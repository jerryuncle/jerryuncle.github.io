Zookeeper API可以实现zookeeper客户端对节点的各种交互，所以我们只需要记住这一个zookeeper()入口即可，对每种方法提供了同步和异步两种方式

  • 同步：有返回值，且需要抛异常
  • 异步：无返回值，不需要关注异常，会通过返回状态码进行判断

Result Code

  • 0 (ok) :接口调用成功
  • -4 (ConnectionLoss) :客户端和服务端连接断开
  • -110 (NodeExists) :节点已存在
  • -112 (SessionExpired) :会话已过期
