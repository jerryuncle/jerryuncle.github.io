二 DataFrame 格式插入

==有时候习惯用pandas模块去读数据库或者excel文件，因为比较方便，但是当把处理结果想写入另外的地方总会出现各种问题==

==思路==

结果转化为dataframe类型
（1）直接通过pandas进行运算生成的dataframe类型
（2）将列表转化为字典，在转化为dataframe类型

==需要插入的数据库连接方式需要用以下方式，读取的数据库或其他文件类型方式不限==
from sqlalchemy import create_engine
import pymysql
engine = create_engine(“mysql+pymysql://user:passwd@ip/db_name”,encoding=‘utf8’)
data.to_sql(‘table_name’,con=engine,if_exists=‘append’,index=False)
