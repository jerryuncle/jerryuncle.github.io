Sqoop 作用：
数据从RDBMS和Hadoop之间进行导入导出操作
底层就是使用MapReduce来实现的
Map 只有Map
Reduce 没有Reduce

==导入: import
RDBMS --> Hdfs
RDBMS–> Hive==

==导出: export
Hdfs–>RDBMS
Hive–>RDBMS*==

参数帮助查看—》sqoop help

exoprt : 导出
import : 导入
job : 作业
list-databases : 数据库列表
list-tables : 表列表

以上就是简单的查看命令帮助，并输出数据库和数据库表相关信息

数据导入HDFS

HDBMS—>HDFS

  • 全表导入

==当执行关系型数据库导入hdfs的时候，默认会在执行目录下产生一个xxx.java的文件==

  • 导入指定列
  • 查询限制导入
  • 增量导入
  • options-file
    sqoop --options-file xxx.opt

==往hdfs中传输数据的时候，因为map默认是4并且按照主键去拆分，当表没有主键的时候：
需要指定–num-mappers 1
或者需要用–split-by ‘column-name’==
